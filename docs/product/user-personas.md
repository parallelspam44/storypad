# User Personas

## Primary Persona: The Minimalist Writer

### Demographics

- **Age:** 18-45
- **Tech Savvy:** Moderate to high
- **Platform:** Android (majority), iOS (growing)

### Characteristics

- Values simplicity and clean design
- Writes regularly (daily notes, journal, thoughts)
- Prefers one-time purchases over subscriptions
- Privacy-conscious
- Appreciates customization options

### Pain Points

- Overwhelmed by complex journaling apps
- Frustrated with subscription models
- Concerned about data privacy
- Wants powerful features without clutter

### Why StoryPad?

- Timeline-based journaling (no folder management)
- Extensive editing options in simple UI
- Privacy-first approach
- One-time add-on purchases
- Open source transparency

## Use Cases

### Daily Journaling

- Quick daily entries on timeline
- Mood tracking with 45+ emotions
- Photo memories with custom layouts
- Throwback memories feature

### Creative Writing

- Multi-page entries for novels/stories
- Rich text formatting (bold, colors, fonts)
- Templates for writing prompts
- Distraction-free writing environment

### Personal Organization

- Tags and stars for organization
- Search functionality
- Multi-page entries for projects
- Backup to Google Drive

### Focused Writing Sessions

- Relaxing sounds add-on for ambiance
- Minimalist interface reduces distractions
- Customizable themes and fonts
- Offline-first design

## Secondary Personas

### The Privacy Advocate

- Strongly values data ownership
- Appreciates open source transparency
- Uses local-first features
- Avoids cloud-dependent apps

### The Customization Enthusiast

- Loves 1300+ Google Fonts
- Experiments with themes and colors
- Creates custom templates
- Enjoys rich formatting options

### The Routine Builder

- Uses templates for consistency
- Tracks moods and emotions
- Interested in upcoming routine features
- Values habit formation tools
